Page Studio
=============

A lightweight PHP CMS built to speed up web development and simplify content 
management.

#Website
http://www.pagestudiocms.com/

#Requirements
A web host or server with:
* PHP >= 5.4.0
* MySQL >= 4.x

A modern and updated web browser

#Documentation
http://pagestudiocms.com/docs/
